📸 Annie Leibovitz — Photography Portfolio

A responsive portfolio website inspired by Annie Leibovitz’s iconic photography style. Showcases emotion-driven portraits and storytelling through a clean, elegant web interface.

Live Demo: [annie-leibovitz.vercel.app(https://annie-leibovitz.vercel.app/index2.html)]

🛠 Technologies Used

HTML5 – For semantic structure and content

CSS3 – For styling, layout, and responsive design

JavaScript – For interactive elements (sliders, animations)

Vercel – For hosting and deployment

Google Fonts / Font Awesome – Typography & icons

Responsive Design Techniques – Mobile-first approach, media queries

✨ Key Features

Clean, modern design highlighting photography

Full-page hero section with featured images

Responsive layout for desktops, tablets, and mobiles

Smooth scrolling and subtle hover animations

Social media integration (Instagram, Twitter, YouTube)

About, Work, Journal, and Contact sections

Footer with studio details and branding

Minimal text to emphasize visuals

⌨ Keyboard Shortcuts & Interactions

Scroll / Arrow keys – Navigate through sections smoothly

Hover on images – Reveal captions or subtle effects

Click on links – Open social media or contact forms

Tab navigation – Accessibility-friendly focus on links/buttons

📝 Development Process

Planning & Sketching:
Outlined sections, content hierarchy, and desired user experience.

HTML Structure:
Built semantic HTML sections (header, section, footer) to organize content.

CSS Styling:
Applied layout styling, fonts, colors, and responsive breakpoints.

JavaScript Integration:
Added smooth scrolling, interactive buttons, and hover effects for gallery images.

Deployment:
Hosted the website on Vercel for live access and sharing.
